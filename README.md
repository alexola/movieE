
# MovieE

Understanding the API calls thru a different aproach from previous projects and now is made with a public Movie API.


## Installation

Just run either the one you prefer .




```bash
    npm install
    npm run 

    yarn install
    yarn run
```

**Client:** React, Axios, CSS
